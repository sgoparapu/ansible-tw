Thought Works Demo

This is demo for setting up a 2 node web server and 1 db node

Infra setup:

2 Web Servers which are launched in 2 different availability zones and 2 different public subnets

1 DB Server  which is hosted in one of the availability zone in a private network

There is communication established from Public facing instances to Private(DB Server)

Both the public facing instances are attached to Application Load Balancer and request are server when the load balancer is hit

This is all done with the help of Terraform and meanwhile webserver needs apache and DB needs database installed so we have used ansible to do this activity and is a masterless setup where it with the help of user_data during the instance launch it clones the repo and runs the playbook

We can further enhance using the real code and having a master ansible host to control the deploymemnts as well
